display - none : 해당 요소를 감출 때 사용(치운다라는 느낌) inline : 컨텐츠의 영역만큼 가져감. inline-block : 기본적으로는 inline과 유사한 특징을 가지지 넓이, 높이 지정이 가능한 속성 visibility : 해당 요소를 보이게 할 것인지 말 것인지를 정의하는 속성 - visible : 해당 요소를 보이게함(기본값) - hidden : 해당 요소를 숨김(감춘다) - collapse : table 요소에 사용하는 속성(행이나 열을 감춤) opacity : 요소의 투명도 결정 0~!까지의 값(o.1, 0.2 ...) - 0 : 완전 투명, 1 : 기본값 보여줄 때 효과를 위해 다른 속성들과 연계해서 사용하기도 하지만 단순한 view의 시각적감각을 표현하기 위해서도 사용

display none vs visibility hidden

- 둘의 차이는 해당 요소의 공간자체를 없애느냐 혹은 공간은 남겨두느냐의 차이라고 볼 수 있다.
